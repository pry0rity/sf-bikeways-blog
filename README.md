## Tech Stack

- [Astro](https://astro.build)
- [Tailwind CSS](https://tailwindcss.com)
- [TypeScript](https://www.typescriptlang.org)

## Project Structure

```
sf-bikeways-blog/
├── public/          # Static assets
├── src/
│   ├── components/  # Reusable components
│   ├── content/     # Blog posts
│   ├── data/        # Route data
│   ├── layouts/     # Page layouts
│   ├── pages/       # Route pages
│   └── styles/      # Global styles
└── package.json
```

## Development

```sh
npm install
```
